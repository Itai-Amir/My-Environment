Privacy Policy – Page Manager (Dj Americano)

Last updated: 2026-02-01

Who we are

This app (“Page Manager”) is operated by Itai Amir for managing the Facebook Page “Dj Americano”.

What data we access

  • Basic profile information (name and profile picture)
  • The list of Pages the operator manages
  • Page engagement data needed for posting (e.g., post IDs created by the app)

What we do with the data

  • Publish posts to the “Dj Americano” Facebook Page
  • Avoid duplicate content (using a local history of what was posted)

What we do NOT do

  • We do not sell data.
  • We do not share data with third parties.
  • We do not use data for advertising targeting.

Data storage and retention

  • Access tokens are stored securely as encrypted secrets in GitHub (GitHub Actions Secrets).
  • A minimal posting history is stored in a file (state/state.json) in the app’s repository to prevent duplicates.
  • Logs may contain technical information but do not store personal messages.

Data deletion

If you want all stored data removed, email: itaamir76@gmail.com
We will remove stored tokens and any stored posting history within a reasonable time.
